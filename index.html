Featured Research

AI / ML • Computer Vision • Scientific Computing

AutoMammo — AI Mammography

Engineered a system to automatically classify and prioritize high-risk mammography cases. Fine-tuned Google MedGemma 1.5 4B-it vision-language model using LoRA and 4-bit quantization. Achieved ROUGE-L: 0.6693, METEOR: 0.7187, and Word-Level F1: 0.6789.

MedGemma VLM LoRA / QLoRA Computer Vision
Repository

Retinal Morphology Analysis

Developed an automated, config-driven 7-step end-to-end ML pipeline for RPE cell classification. Extracted 133 morphological/texture features and achieved 90%+ cross-validation F1-score using a stacking ensemble (XGBoost, LightGBM, CatBoost) with a Logistic Regression meta-learner.

Ensemble ML Feature Engineering OpenCV Scikit-Image
Repository

PINNs for Cancer Invasion

Engineered a PINNs model to simulate collective cancer invasion as a data-efficient, mesh-free solution for complex PDEs. Utilized Tensorized Fourier Neural Operators (TFNO) and explored Seq2Seq PINO and Augmented Lagrangian methods to improve solution accuracy.

PINNs PyTorch PDEs TFNO
Repository

Neuromorphic Biocomputing

Simulating biological "wetware" logic gates using FitzHugh-Nagumo and Hodgkin-Huxley models. Demonstrated how neural signaling dynamics perform Boolean AND operations through phase space trajectory analysis and coincidence detection.

Neural Simulation Diff. Equations Biophysics Manim

Quant Trading Lab — Financial Engineering

A comprehensive collection of quantitative finance projects implemented in Python, covering stochastic simulations (GBM, OU), derivatives pricing (Crank-Nicolson PDE), and volatility calibration. Implemented stat-arb strategies including pairs trading based on cointegration.
