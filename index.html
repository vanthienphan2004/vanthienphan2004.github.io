Neuromorphic Biocomputing

Simulating "wetware" logic gates using FitzHugh-Nagumo and Hodgkin-Huxley models. Demonstrated how signaling dynamics in FHN neurons can perform Boolean AND operations through phase space trajectory analysis and coincidence detection.

PINNs for Cancer Invasion

Developed a Physics-Informed Neural Network (PINN) framework to simulate collective cancer invasion. Integrated Tensorized Fourier Neural Operators (TFNO) to solve coupled PDEs governing tumor growth and matrix degradation with high data efficiency.

Retinal Morphology Analysis

Engineered an end-to-end ML pipeline for classifying aging Retinal Pigment Epithelial cells. Extracted 133 morphological features and achieved >94% F1-score using a stacking ensemble of XGBoost and LightGBM models.
