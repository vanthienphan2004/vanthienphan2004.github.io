Morphological Feature Analysis of Retinal Pigment Epithelial Cells

A configuration-driven Python application for extracting, aggregating, and analyzing morphological features from Retinal Pigment Epithelium (RPE) image crops. The pipeline produces cleaned feature tables, trains a stacking ensemble classifier (XGBoost + LightGBM + CatBoost), performs feature importance analysis, and generates reproducible model artifacts.

Key Features:

  • 133+ morphological features including shape, intensity stats, texture (LBP, GLCM, Gabor)
  • Automated 7-step pipeline: extraction → cleaning → training → analysis
  • Stacking ensemble with cross-validation
  • Feature importance analysis and visualization

Technologies: Python, scikit-learn, XGBoost, LightGBM, CatBoost, OpenCV, scikit-image

Physics-Informed Neural Networks for Collective Cancer Invasion

A comprehensive implementation of physics-informed neural networks (PINNs) and Fourier neural operators (FNOs) for modeling collective cancer invasion dynamics. This project solves coupled nonlinear partial differential equations describing the spatiotemporal evolution of leader and follower cancer cell populations using advanced machine learning techniques.

Key Features:

  • Physics-informed learning with PDE constraints
  • Fourier Neural Operators for efficient PDE solving
  • Coupled reaction-diffusion equations modeling
  • Advanced visualization and multi-format results

Technologies: Python, PyTorch, NumPy, Matplotlib, CUDA
